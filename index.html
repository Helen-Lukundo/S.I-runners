# S.I Runners A neon-cyberpunk endless runner that plays in any browser — touch on mobile, keyboard on desktop. ## Play locally Just open `index.html` in a browser. That's it (it's one self-contained file). ## Controls - **Desktop:** `SPACE` / `↑` to jump, `↓` to slide - **Mobile:** swipe up to jump, swipe down to slide (or use the on-screen buttons) ## Push to GitHub ```bash cd si-runners git init git add . git commit -m "S.I Runners v1 - playable core" git branch -M main git remote add origin https://github.com/YOUR_USERNAME/si-runners.git git push -u origin main ``` ## Deploy on Render (free) 1. Go to https://render.com → **New +** → **Static Site** 2. Connect your GitHub repo (`si-runners`) 3. Settings: - **Build Command:** *(leave empty)* - **Publish Directory:** `.` 4. Click **Create Static Site**. Render gives you a public URL that updates every time you `git push`. ## What's in v1 - Running, jumping, sliding with gravity/physics - Three obstacle types: crystals (jump), barbed wire (slide under), drones (slide under) - Coin pickups, score, rising difficulty, saved high score - Parallax neon city, particles, sound effects, sound toggle - Fully responsive + works with touch and keyboard ## Roadmap (next features) - [ ] Character select (cyberteens + normal runners) - [ ] Weapons & combat (swords / guns / shields vs monsters) - [ ] Boosts: speed, shield, invisibility, strength, resurrection, amulets - [ ] Chests with rewards - [ ] Background themes: day / doomsday (night is in now) - [ ] Phonk music tracks with mute
